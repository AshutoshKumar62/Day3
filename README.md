Project Overview

This project is a simple RESTful CRUD API built using Node.js and Express.js.

It allows users to:

 View all books

 Add a new book

 Update an existing book

 Delete a book

The data is stored in an in-memory array (no database used).

 Features

Express server running on Port 3000

RESTful API endpoints

JSON request & response handling

In-memory data storage

Tested using Postman
